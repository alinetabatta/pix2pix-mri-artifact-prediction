## 📊 Dataset Access

The dual-stage framework utilizes an open-access multi-modal dataset pairing localized sagittal scout views with corresponding target Proton Density Fat-Suppressed (PDFS) clinical sequences.

* **YOLO Localization Dataset (`MetalDetecMRI-v1`):** The annotated bounding-box dataset used to train the lightweight localization network for metallic implant detection is hosted openly on Roboflow. You can download it directly in various formats (YOLOv8, COCO, etc.) via the following link:
  👉 **[Download MetalDetecMRI-v1 on Roboflow](https://app.roboflow.com/tabatta/yolo_sag_m21/1)**
